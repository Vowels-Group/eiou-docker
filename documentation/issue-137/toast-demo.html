Toast Notification System Demo

Interactive demonstration of error handling and toast notifications

1. Basic Toast Notifications

Click buttons to show different types of toast notifications:

Toast.success('Operation completed successfully!');
Toast.error('An error occurred. Please try again.');
Toast.warning('Warning: This action cannot be undone.');
Toast.info('Processing your request...');

2. Multiple Toasts (Stack Test)

Test multiple toasts stacking vertically (max 5 visible):

for (let i = 0; i < 10; i++) {
  Toast.info('Message ' + (i + 1));
}
// Only 5 toasts will be visible (oldest removed)

3. Custom Timeouts

Control how long toasts stay visible:

Toast.success('Quick message!', 2000);  // 2 seconds
Toast.error('Important!', 10000);      // 10 seconds
Toast.warning('Persistent', 0);        // No auto-dismiss

4. Error Handler (HTTP Status Codes)

Simulate different HTTP error responses:

const message = ErrorHandler.getUserFriendlyMessage(400);
// Returns: "Invalid request. Please check your input and try again."

5. Retry Logic (Exponential Backoff)

Simulate network errors with automatic retry:

// Automatically retries on transient errors (500, 502, 503, 504)
await RetryHandler.retryFetch(() => fetch('/api/endpoint'), {
  maxAttempts: 3,
  initialDelay: 1000,
  backoffMultiplier: 2
});
