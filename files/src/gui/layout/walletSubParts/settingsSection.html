Reset to Defaults

Throws every saved wallet setting back to the values this node ships with. Covers the main Wallet Settings fields and every Advanced Settings category (Feature Toggles, Currency, Display, Backup & Logging, Data Retention, Sync, Network, Rate Limiting, GUI Security).

Caveat: “Defaults” means whatever this build of the code considers default. If an operator has edited Constants.php on this node, this resets to the operator’s edited values, not the upstream ship defaults. Your contacts, transactions, backups, and API keys are not touched.

This is also exposed via the CLI — eiou changesettings reset --yes performs the same operation from inside the container.
