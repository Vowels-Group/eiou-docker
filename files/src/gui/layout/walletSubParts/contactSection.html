Pending Contact Requests

About Pending Contact Requests
Incoming requests from other wallets that want to connect with you. Accepting creates a bilateral contact record (and adds any currencies they use if they aren't already in your Allowed Currencies list). Rejecting silently drops the request — the other side isn't told. Outgoing requests you've sent that are still waiting on the other party also live here until they respond.

Subnote on currencies. If a request carries a currency outside your Allowed Currencies list, you'll see an explicit per-currency accept toggle so you can pick which to allow rather than accepting everything blindly. The default behaviour (accept what the contact offers) can be changed under Settings → Feature Toggles → Contacts → Auto-reject Unknown Currency.
