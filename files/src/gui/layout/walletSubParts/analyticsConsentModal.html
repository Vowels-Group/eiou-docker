Help Improve eIOU

Would you like to share anonymous usage statistics to help improve eIOU?

How it stays private

  • Routed entirely through Tor — your IP is never exposed
  • Anonymous ID is a one-way hash — cannot be reversed to your identity
  • No personal data, individual transactions, contacts, or addresses
  • Sent once per week — not continuous tracking

What is sent

  • Aggregate transaction counts (sent, received, relayed)
  • Total volume per currency
  • Number of contacts and active days
  • Node version
